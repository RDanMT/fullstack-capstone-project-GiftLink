# StayHealthy - Doctor Appointments Application

This is the Capstone Project for the IBM Full-Stack Software Developer course.

## Overview
StayHealthy is a complete full-stack web application designed to allow users to book appointments with doctors online, manage their profiles, leave reviews for doctors, and request instant video consultations.

## Features
* **Authentication**: Users can sign up, login, and manage their session locally.
* **Find a Doctor**: Patients can search for doctors by specialty and book appointments.
* **Instant Consultation**: Get an instant consultation through a mock video-call interface.
* **Reviews**: Provide and view reviews for different doctors you've interacted with.
* **Profile Management**: View and modify your patient profile including name and contact info.
* **Notifications**: Get notified of your latest booked appointments upon login.

## Technologies Used
* **Frontend**: React.js, React Router DOM, Vite
* **Styling**: Native CSS, Responsive Design

## How to Run Locally
1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Build for production: `npm run build`
